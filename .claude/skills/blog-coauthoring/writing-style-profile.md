# Xianying（谭显英）博客写作风格画像

> 基于 shrektan.com 约10篇中文博文的深度分析（2018-2025），涵盖个人反思、技术分享、思维随笔、正式观点文章四类。

---

## 一、整体定位

用户的博客是**真实的个人表达空间**——不是营销号、知识付费或公众号爆文。

- **一个聪明人在自言自语**——有时写给自己备忘，有时写给同频读者
- **理工科底子 + 人文触觉**——既能写技术架构分析，也能写深层心理剖析
- **真诚到近乎赤裸**——尤其个人反思类，毫不回避自己的弱点和挣扎

---

## 二、四种文章类型

### A. 个人反思

代表作：《小彻小悟（一）（二）》《不争就输了》《冰水混合物的温度为什么是零》

- 从**具体触发事件**起笔（春节假期、一瓶冰水、一次争吵），自然过渡到深层思考
- 大量**自问自答**——"我常常在想""为什么就会是我？""这有错么？话没错，但是动机错了"
- 情绪流动自然，不刻意煽情也不刻意克制，像在纸上思考
- 偶尔**戏谑自嘲收尾**——"看来，我还是太笨了！""先写到这里……睡了"

### B. 技术分享

代表作：《从 R Shiny 到 FastAPI + React》《自我陶醉》《UTF-8万岁》

- 通常有**生活化引子**（五一假期孩子生病、收到微信群文件），然后切入技术正题
- 技术观点鲜明有立场——会明确说"我建议""我认为""直观感受是"
- 善用**对比结构**呈现技术演进
- 技术讨论中夹带个人经历和感性判断，不会变成纯技术文档

### C. 思维随笔

代表作：《内卷的根源》《恒顺众生》《创造X的价值，捕获Y的比例》《道理与证明》

- 篇幅精炼（300-800字），观点集中
- 常从**外部触发**（YouTube视频、朋友圈文章、讲座）引出观点
- 善于把抽象概念落地到具体例子（iPhone设计、墨菲定律、水的流动）
- 结尾常留有余味——"本想举个栗子，然而时间不早了，得先睡了"

### D. 正式观点文章

代表作：《反馈即进化》

- 核心是**系统性思想洞察或理论框架**
- 开头直接从命题切入或用极简的认知变化引入
- **层层递进**的论证结构，思想密度高
- 语言比随笔类更凝练正式，但仍保持个人博客的温度
- 善于整合不同领域的知识框架（进化论、波普尔哲学、AI 训练等）
- 常用脚注标注来源，增加论证可信度

## 三、语言风格

### 3.1 句式特点

**反问句推进思考：** "真的是必要的么？真的是无法避免的吗？""难道套上一层数学的皮之后就会变得精确了吗？"

**频繁转折：** 几乎每个观点后面跟着"但是/然而/不过"——用户思维天然具有辩证性。

**短句独立成段做强调：** "我也是人。""我能停下来了。"——一句话一段是标志性手法。

**省略号做思绪延伸：** "无尽的会议，激烈的沟通，长期得不到休息，两周的几乎未眠……"

**破折号克制使用：** 偶尔用于补充说明或情绪插入，但不是高频工具。更多时候用逗号从句或括号。

### 3.2 用词偏好

- **口语词是点缀，不是基底**——"叽里咕噜""嘚瑟""鼓捣"等偶尔出现调味，整体是自然书面语
- **四字词不堆砌**——"披荆斩棘""豁然开朗"会用，但不密集
- **中英自然混用**——技术名词保留英文，不生硬翻译
- **emoji 极克制**——通常只在文末用一个做轻松收尾
- **"我"字高频**——文章非常个人化，第一人称贯穿全文

### 3.3 标点习惯

- 引号频繁——给关键概念、引申义、反讽加引号：如"开窍""悟""创造性"
- 标准中文标点
- 喜欢用括号做补充——"（可能的）答案""（至少在学校的教学里）"

### 3.4 段落节奏

- `###` 做大段落间话题转换
- 重要判断独立成短段落（一句话一段）
- 段落长短不均匀——长段展开，短段转折，形成呼吸感
- 过渡靠思路的内在逻辑，不用路标句

## 四、结构模板

### 个人反思类

```
[生活化引子 / 触发事件]
[描述痛点/困境，大量具体细节]
[自问自答，层层深入]
[插一段"题外话"或自嘲，缓解情绪]
[核心洞察/领悟]
[收尾：留有余味，或戏谑式]
```

### 技术分享类

```
[生活化引子]
## 背景与历史回顾
## 问题分析（鲜明观点 + 对比结构）
### 具体问题拆解（加粗编号要点 + 展开）
## 新方案 / 建议
## 结论（简洁有立场）
```

### 思维随笔类

```
[外部触发 + 引用]
[引出核心观点]
[2-4个具体例子佐证]
[递进式或开放式收尾]
```

### 正式观点文章

```
[极简开头：认知变化或直接抛出命题]
### 具体观察（1-2段，点到为止）
### 被忽视的前提（揭示为什么低估了某个概念）
### 理论框架引入（跨领域整合，脚注标注来源）
### 核心论证
### 世界观层面的对比（两种对立看法，揭示深层差异）
### 落地到行动（可选）
### 收尾（回扣命题，简洁有力）
```

## 五、写作禁忌

1. 不写公众号体——"震惊！""建议收藏！"
2. 不端着——不用官方腔、学术腔
3. 不面面俱到——"偏题"恰恰是魅力所在
4. 不刻意煽情——情绪自然流露
5. 不过度格式——标题不超三层，不大量加粗
6. 不隐藏立场——鲜明个人判断是最大特点
7. 不回避自嘲——适度幽默是调味剂
8. 不写管理咨询腔——"闭环""赋能""抓手"
9. 不在文章中解释文章——"接下来我将从三个角度……"
10. 不替用户定义内心状态——涉及心态动机，宁模糊勿武断
